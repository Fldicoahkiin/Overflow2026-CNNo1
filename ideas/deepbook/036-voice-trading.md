# Voice Trading / 语音驱动交易

> 语音指令下单："买入 BTC 上涨到 7 万，15 分钟，100 USDC"

## 解决什么问题

在开车、做饭或运动等无法手动操作手机的场景中，交易者无法及时把握市场机会。Voice Trading 支持语音指令下单，用户只需说出交易意图（如"买入 BTC 上涨到 7 万，15 分钟，100 USDC"），系统自动解析语音为结构化交易参数并执行。语音交易解放双手，让预测交易更加便捷和无障碍。

## 核心功能

- 语音识别与解析：将自然语言语音转为结构化交易参数
- 多语言支持：中文/英文语音指令识别
- 语音确认：执行前语音播报交易摘要，等待确认
- 语音播报：定时播报 BTC 价格和持仓盈亏

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::mint | 解析语音后执行交易 |
| predict::redeem_permissionless | 语音指令赎回仓位 |
| PredictManager | 管理用户仓位和资金 |
| OracleSVI | 语音播报实时价格 |

## 实现方案

### Bot / Keeper / 服务端
- 语音识别引擎：Whisper API / Web Speech API 转文字
- NLP 解析器：从文字中提取交易参数（方向、行权价、金额、到期日）
- 参数验证器：检查参数合理性（行权价范围、金额上限）
- 交易执行器：确认后调用 predict::mint
- 语音合成器：TTS 播报交易结果和行情

### 前端 / 后端
- 技术栈：React + Web Speech API + Node.js
- 语音交互界面：麦克风按钮、实时转写显示
- 交易确认卡片：解析结果展示，语音/按钮确认
- 语音设置面板：语言选择、播报频率、快捷指令配置

## 技术架构

用户按下麦克风说话 → 语音识别引擎转文字 → NLP 解析器提取参数 → 参数验证器检查 → 展示确认卡片并语音播报 → 用户确认 → predict::mint 执行 → 语音播报结果。
