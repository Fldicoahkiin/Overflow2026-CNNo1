# PLP Risk Dashboard / PLP 风险仪表盘

> 金库利用率、提款限制器状态、极端场景模拟（±5σ BTC 移动）

## 解决什么问题

PLP（Predict Liquidity Provider）是 Predict 的核心流动性层，LP 需要全面了解金库风险状况——包括利用率、提款限制器状态和在极端市场情景下的潜在损失。PLP Risk Dashboard 为机构级 LP 提供专业风险管理仪表盘，展示金库利用率、每行权价风险敞口、提款限制器令牌桶状态，以及 ±5σ BTC 价格移动下的压力测试结果。

## 核心功能

- 金库利用率监控：PLP 总供应量 vs 已使用量
- 提款限制器令牌桶状态：剩余容量和恢复速率
- 极端场景模拟：±1/3/5σ BTC 移动下的 PLP PnL 影响
- 每行权价风险敞口热力图

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::supply | 理解 PLP 供应和收益机制 |
| 提款限制器 | 读取令牌桶参数和状态 |
| OracleSVI | 计算各行权价的理论敞口 |
| 链上仓位数据 | 分析 PLP 实际风险分布 |

## 实现方案

### Bot / Keeper / 服务端
- 链上数据采集器：定期读取 PLP 金库状态和仓位分布
- 风险计算引擎：基于当前仓位计算各行权价风险敞口
- 压力测试模拟器：计算 ±Nσ BTC 移动下的预期损失
- 令牌桶模拟器：实时推算提款限制器状态
- 报告生成器：生成可导出的风险评估报告

### 前端 / 后端
- 技术栈：React + ECharts/D3.js + Python FastAPI
- 风险仪表盘：利用率、敞口分布、令牌桶状态
- 压力测试面板：滑块调节 BTC 移动幅度，实时显示 PnL
- 热力图：行权价 × 到期日的风险敞口分布
- 历史回撤回放：展示 PLP 历史净值曲线

## 技术架构

链上数据采集器读取 PLP 状态 → 风险计算引擎分析敞口分布 → 压力测试模拟器计算极端情景 → 令牌桶模拟器推算提款状态 → 前端仪表盘实时展示 → 报告生成器定期输出。
