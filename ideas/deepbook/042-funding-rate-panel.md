# Funding Rate Panel / 实时资金费率面板

> 显示各到期日的隐含利率和资金费率

## 解决什么问题

Predict 市场中不同到期日的定价隐含了不同的利率水平，但缺少直观的工具展示这些隐含利率和资金费率。交易者和 LP 需要理解各到期日的定价偏差（相对于理论值）所隐含的利率，以做出更明智的持仓和流动性决策。Funding Rate Panel 实时计算并展示各到期日的隐含利率和资金费率。

## 核心功能

- 各到期日隐含利率实时计算和展示
- 资金费率趋势图：历史费率变化和均值回归分析
- 跨期比较：近月 vs 远月隐含利率差异
- 费率异常告警：当隐含利率偏离正常范围时通知

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| OracleSVI | 计算理论价格作为费率基准 |
| 链上 mint 事件 | 观察实际定价偏差 |
| predict-server API | 获取各到期日市场数据 |
| OracleSVIUpdated 事件 | 实时更新费率计算 |

## 实现方案

### Bot / Keeper / 服务端
- 定价计算器：基于 SVI 参数计算各区间的理论价格
- 隐含利率提取器：从实际交易价格与理论价格偏差中提取隐含利率
- 费率分析器：统计各到期日的费率趋势和均值
- 异常检测器：识别费率偏离正常范围的异常
- 历史存储：记录费率变化时间序列

### 前端 / 后端
- 技术栈：React + ECharts + Node.js
- 资金费率面板：各到期日费率实时展示
- 趋势图表：历史费率曲线和移动平均
- 跨期对比：近远期费率差异柱状图
- 告警配置：自定义费率阈值通知

## 技术架构

OracleSVI 提供理论定价参数 → 定价计算器计算理论价格 → 链上 mint 事件提供实际定价 → 隐含利率提取器计算偏差 → 费率分析器统计趋势 → 前端展示实时和历史费率。
